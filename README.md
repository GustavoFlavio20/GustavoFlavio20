# 👨‍💻 Gustavo Almeida Flavio

**`Consultor Júnior de Integração`**

👋 Olá! Sou o Gustavo Almeida Flávio, tenho 24 anos e sou graduado em Análise e Desenvolvimento de Sistemas.

Atualmente atuo como Desenvolvedor Júnior de SAP Integration Suite, desenvolvendo integrações no ecossistema SAP Business Technology Platform (SAP BTP). 
Tenho interesse em integração de sistemas, APIs, automação de processos e soluções empresariais, buscando unir conhecimento técnico e visão de negócios.
<br>
<br>
Neste GitHub compartilho projetos desenvolvidos durante meus estudos e experiências práticas, utilizando tecnologias como SAP Integration Suite (Cloud Integration), Groovy, XML, JSON e integrações com APIs REST.
<br>
